# dir-util

```
Usage: targetDirectory [option]
    -t, --targetDirectory=           Specify target path (default:.)
    -o, --sortOrder=                 Specify normal or reverse (default:normal)
    -s, --sortRule=                  Specify dir-name or file-name (default:dir-name)
    -n, --numOutput=                 Specify the number of output result (default:1)
    -m, --outputMode=                Specify output mode: relative or full (default:relative)
    -f, --filterMatch=               Specify regexp match condition (default:)
```
